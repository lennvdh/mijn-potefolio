Project: Portfolio Website (HTML, CSS & Bootstrap)

Beschrijving

Voor dit project heb ik een persoonlijke portfolio-website gemaakt met behulp van HTML, CSS en Bootstrap. Het doel van deze website was om mezelf online voor te stellen als web developer en mijn projecten op een overzichtelijke en professionele manier te presenteren.

Doel van het project:

Het ontwikkelen van een responsive portfolio website die goed werkt op zowel desktop, tablet als smartphone, en waarin mijn vaardigheden en projecten duidelijk worden weergegeven.

Gebruikte technologieën

  • HTML5 voor de structuur van de website
  • CSS3 voor de styling en visuele aanpassingen
  • Bootstrap 5 voor het responsive grid systeem en UI-componenten

Functionaliteiten:

  1. Een navigatiebalk met smooth scroll naar verschillende secties
  2. Een hero-sectie met een korte introductie
  3. Een “Over mij”-sectie met persoonlijke informatie en skills
  4. Een projecten-sectie met kaarten om mijn werk te tonen
  5. Een contactformulier voor bezoekers

Wat ik heb geleerd:

Tijdens dit project heb ik geleerd hoe ik Bootstrap efficiënt kan gebruiken om snel een moderne layout op te bouwen. Ook heb ik mijn kennis van HTML en CSS verbeterd, vooral rond structuur, spacing en responsive design.

Conclusie:

Dit project heeft mij geholpen om beter te begrijpen hoe een professionele portfolio-website wordt opgebouwd en hoe belangrijk gebruiksvriendelijkheid en design zijn in webontwikkeling.
