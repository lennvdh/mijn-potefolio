Project: Active Directory Server opzetten en configureren

Beschrijving

Voor dit project heb ik een Active Directory server opgezet met behulp van Windows Server. De server beheert gebruikers, computers en toegangsrechten binnen een netwerk. Het doel was om een gestructureerde en veilige IT-omgeving te creëren zoals in een echte bedrijfsomgeving.

Doelstellingen

  • Installeren en configureren van een domeincontroller
  • Gebruikers en groepen beheren
  • Policies (GPO’s) instellen
  • Netwerkstructuur organiseren
  • Beveiliging verbeteren

Gebruikte technologieën

  • Windows Server
  • Active Directory
  • Group Policy
  • VirtualBox (of VMware indien gebruikt)

Werking van het systeem

De server functioneert als centrale beheerder van het netwerk:

  1. Installatie van Windows Server
  2. Configuratie als Domain Controller
  3. Opzetten van een domein (bijv. school.local)
  4. Toevoegen van gebruikers en computers
  5. Toepassen van Group Policies

Resultaat

De Active Directory server werkt als centrale beheeromgeving waarin:

  • gebruikers eenvoudig beheerd worden
  • beveiligingsregels automatisch toegepast worden
  • computers binnen het domein gecontroleerd worden

  • PowerShell gebruiken voor automatisatie
